## Загальні висновки по алгоритмам пошуку:
### 1. Алгоритм пошуку Боєра-Мура:
1. Краще працює, коли шуканий текст знаходиться ближче до кінця тексту для пошуку
2. Збільшення патерну пошуку прискорює пошук
3. Найбільш висока швидкість при негативному пошуку
### 2. Алгоритм пошуку Кнута-Морріса-Пратта:
1. Краще працює при пошуку тексту на початку текстового масиву
2. Чим більше паттерн тексту для пошуку, тим швидший пошук
3. Низька швидкість при негативному результаті пошуку
### 3. Алгоритм пошуку Рабіна-Карпа:
1. Швидкість пошуку залежить від розташування шуканої строки в тексті. Чим ближче до початку, тим швидше
2. Збільшення паттерну пошуку уповільнює результат пошуку
3. Ймовірно прийнятним буде при використанні стабільних патернів пошуку в завчасно порахованих хеш-значеннях
